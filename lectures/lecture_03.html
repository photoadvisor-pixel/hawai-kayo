2. 視覚化の3段階訓練法：目的と効果

視覚の重要性

佳代：視覚は5つの感覚（視覚、聴覚、味覚、触覚、嗅覚）の中で最も強力な働きをします。目から入ってくる情報は自分自身に大きな影響を与えます。だから何を見るかを自分でチョイスしていかないといけない。

寝る前の注意：変なニュースを見て寝ると、そのネガティブなエネルギーに同調した世界に寝ている時に行ってしまう。寝る前はエネルギーを整えて、周波数を高くした状態で眠ることが大切。

音の力について

佳代：聴覚の力も大きい。音楽は脳波をβ波からα波、そしてθ波に導くのに利用できます。

ただし注意点があります。音を作る人の周波数がそのまま反映される。自我を超越した意識で紡いだ音と、そうでない音は違う。感情に訴える音はまだアストラルレベルの周波数。

高い意識から出る音は感情を超越しています。感情を超えた時に流れる涙を「聖なる涙（Divine Tears）」と呼びます。これは感情の浄化の涙とは質が違います。瞑想で高い境地に行きたいなら、そういう高い周波数の音、あるいは無音の中に身を置くことが大切です。

視覚化訓練の目的

  • ハイヤーセルフからのメッセージをビジョンで受け取る
  • ヒーリング
  • チャクラの開発
  • クンダリーニの覚醒

心の目で見る感覚を養う：これが第3の目の活性化につながります。ただし浄化・癒しがなされていないと変な方向に行く危険がある。うちの瞑想では自然な形で活性化されるので安全です。

3. 第1ステップ：記憶像の再現

目的

目で見たものを目を閉じてもイメージとして再現できるようにする。

練習1：物体のイメージ

  1. コップやペンなど身近なものを1つ選び、目の前に置く
  2. 1分間じっくり観察する（形、細部、色など）
  3. 目を閉じて、今見たものをイメージする
  4. 色や形がイメージできれば成功

ポイント： - 頭で考えてイメージしようとしない - 当たり前にビジョンがイメージできると確信する - 落ち着いた心で見る - 淡々とやる（子供が単語を覚えるように）

Q&A：うまくイメージできない時は？

Q： 目を開けて見た時のようなクリアな形には見えず、ぼわっとして形が定まりません。周りの風景も含めてイメージすべきでしょうか？

A（佳代）： 最初はぼわっとしていても大丈夫です。「できた」という感覚、成功体験を持つことが入口になります。 コツは「シンプル」にすること。あれこれ考えずに「ただコップだけ」をパッとイメージする。淡々と、子供が絵を見て言葉を覚えるように、直感的にパパッとやっていくのが脳の回路を開くコツです。考えすぎると左脳が働いてしまうので、右脳の感覚で遊ぶようにやってみてください。

練習2：色のイメージ

虹の7色＋白、金色をイメージする練習： - 赤：りんご、イチゴ - 橙：みかん、ニンジン - 黄：レモン、ひまわり - 緑：キュウリ、ピーマン、葉っぱ - 青：青空、青いインク - 紫：ナス、桔梗の花 - 白：雲、雪、牛乳 - 金：金貨

4. 第2ステップ：イメージの変化

練習1：大きさ・色・形を変える

大きさ：赤いりんごをイメージし、巨大にしたり、手のひらサイズに小さくしたりする。

色：赤いりんご → 青りんご → 黄色 → 緑 → 紫 → 金色 → 白

形：普通のりんご → 三角形 → 四角形 → 台形 → 六芒星 → ピラミッド型

練習2：動きのあるイメージ

  • 人が走っている様子
  • 車が通り過ぎる
  • 電車が目の前を通過する
  • 風車が回っている
  • お茶を急須から湯呑みに入れる
  • 花火が上がっていく

余談：花火が上がっている時、みんな喜んでいる。これはアセンションのエネルギー。3次元から4次元へのシフト（アセンション）は喜びのエネルギー。

さらにその上、5次元意識は「無条件の愛」に目覚めることです。 例えば、富士山に大砲を撃ち込まれたとしても、富士山はそれさえも受け入れる。それが無条件の愛の境地です。アセンション（4次元）ではまだ日常のアップダウンがあるかもしれませんが、5次元意識に行くと、全てを受け入れ、愛することができるようになります。私たちが目指すのはそこです。

5. 第3ステップ：イメージの投影と強化

練習1：目を開けたままのイメージ

  1. 目を閉じてレモンを思い浮かべる
  2. 目を開けて、テーブルの上にレモンがあるのをイメージする

佳代：こういう風に目を開けてもあるのが感じられると、他の次元とのコミュニケーションができるようになる。今も他の次元の存在たちが来ているのが、目を開けていても見える。次元が重なって見える。

練習2：人物のイメージ

家族、友人、知人の中から誰かを選び、その人の様々な表情をイメージする： - 笑っている - 怒っている - 泣いている - 元気いっぱい

さらに感情の度合いも感じ取る。

練習3：自己のイメージ（俯瞰視点）

日常生活をしている自分を客観的にイメージする。

2つの視点の違い： - 同じ部屋にいる視点：現実的なこと、細かいことが気になる - 地球の外からの視点：意識が拡大し、小さなことはどうでもよくなる

佳代：同じ部屋にいると「もっと片付けをやればいいのに」と見える。地球の外から見ると「もう十分今まで頑張ってきたからちょっと休憩したら」と感じる。このコントラストがハイヤーセルフの意識と肉体次元の意識の差。

究極のシンプルさ：自分の中心＝宇宙の中心

激動の時代において、いろいろなものを抱え込みすぎると苦しくなります。地球の外から自分を見る視点は、「自分の中心＝宇宙の中心」という感覚に繋がります。 本当はそれだけで全てであり、とてもシンプルなんです。不安からいろいろなものを抱え込んで難しくしてしまっているだけ。不要なものを削ぎ落とし、シンプルに生きることで、本来の魂の目的に沿った生き方ができるようになります。

6. 複数のエネルギー体

ヨガ哲学の5つの体（パンチャ・コーシャ）

  1. 肉体（アンナマヤ・コーシャ）
  2. 食べ物によって維持される体

  3. 五感で認識でき、科学的に測定可能

  4. エーテル体（プラーナマヤ・コーシャ）

  5. 生命エネルギー（プラーナ、気）を司る
  6. チャクラや経絡が存在
  7. エーテル体が先で、肉体はその青写真に基づく
  8. 瞑想や気功で気のボディを整えると、後から体の調子が良くなる

  9. 肉体のすぐ外側、数cm〜数十cmの幅で広がる

  10. アストラル体（マノマヤ・コーシャ）

  11. 感情や思考、潜在意識の領域
  12. 夢を見たり幽体離脱が起こる層

  13. 人間が死ぬとすぐアストラル界に行く

  14. メンタル体（ヴィジュナーナマヤ・コーシャ）

  15. 知性や直感を司る

  16. 高次の意識、ハイヤーセルフとつながりやすい

  17. コーザル体（アーナンダマヤ・コーシャ）

  18. 魂の本質や霊的な記憶を持つ最も高次の体
  19. 至福の体とも呼ばれる
  20. 悟りや自己実現と関連
  21. 宇宙意識との一体感を持つ領域

ハイヤーセルフについて

ハイヤーセルフとは、宇宙意識から初めて個性を持った存在。あなたにとっての最高の先生であり、自分自身でもある。

瞑想で整えていくことによって全ての体も整い、ハイヤーセルフとの繋がりが深くなり、宇宙の根源につながれる。

7. 肉体の覚化：目的と効果

目的

肉体の感覚器官を通じないで、物質の背後にある精妙な波動をキャッチできるようになること。

佳代：私がいつもやっていること。何か物事が起きても、それを見るのではなく背後のエネルギーを読み解く。そちらの方が正確に物事が見れる。

練習1：手を合わせる感覚

  1. 目を閉じて手を胸の前で合わせる
  2. その時の肉体感覚を覚える
  3. 手を下ろして、合わせていた時の感覚を想像する
  4. リアリティを感じることができたら成功

練習2：座る・立つの感覚移動

重要な練習：ハイヤーセルフの意識とコンタクトするのに非常に重要。

  1. 座って肉体感覚を記憶する（意識は頭にある）
  2. 立つ
  3. 座っていた時の頭の位置が、立った時に胸の辺りにあるのを感じる
  4. 意識が胸で感じられるか確認

佳代：普段私たちの意識は頭にある。でもハイヤーセルフの意識は胸にある。だから意識を胸に下ろさないといけない。この練習はそのためのもの。

練習3：体を傾ける

  1. 楽な姿勢で座り、体全体の知覚に意識を向ける
  2. 10秒かけて体を右に傾ける
  3. 10秒かけて左に傾ける
  4. これを繰り返す
  5. 体を止めても、イメージの中で運動が続いているのを感じる

目的：意識を体から外す練習。肉体意識を超越した意識領域に入っていくため。

練習4：体に任せる運動

  1. 立って、体が一番ほぐれるように勝手に動くとイメージする
  2. 体の知恵を信頼し、頭で考えずに任せる
  3. 音楽をかけながらやると効果的

佳代：体がほぐれると瞑想に入りやすくなる。瞑想とはいかにリラックスするか。心と体は連動しているので、体をほぐすと心もほぐれる。

8. 視覚化の応用：イメージの総合練習

感覚を加える効果

視覚に他の感覚も加えることでリアリティが強まり、肉体レベルまで働きかける作用が大きくなる。感情を付加することでイメージは生体化学反応を引き起こす。

練習1：レモン・梅干しのイメージ

口に入れるのを想像するだけで唾液が出る（パブロフの条件反射）。

練習2：温泉のイメージ

お気に入りの温泉を思い浮かべ、温泉の匂い、お湯の温度、蒸気、景色など五感でイメージする。

佳代：リラックスして素直な心持ちでやると、本当にお風呂に入ったのと同じ効果が得られる。

練習3：過去の体験のイメージ

失敗例と成功例を思い浮かべ、その時の肉体的反応を感じる。

目的：エネルギーを読み解く時、相手の状態が自分の肉体反応として感じられることがある。感覚を研ぎ澄ます訓練。感じる感性は右脳の体験であり、瞑想は右脳の体験。

9. 幽体離脱の訓練

グラウンディング

  1. 尾底骨からコードを地球の中心に伸ばすイメージ
  2. 地球の中心にフックをかけてぐるぐる巻きにする
  3. これで安全に幽体離脱して戻ってこれる

イメージの中での外出

  1. イメージの中で立って玄関まで歩く
  2. 靴を履いて外に出る
  3. 家の周りの景色を細かく思い浮かべる
  4. 着ている服、体の重さ、歩いている感覚を感じる
  5. 自由にあちこち出かける（近所の公園、カフェ、遠出も可）
  6. 人と会ったら会話する

戻ってくる

  1. 家に向かって帰る
  2. 玄関の前で立ち止まる
  3. 部屋の中に入り、座っている自分に近づく
  4. イメージの自分と肉体の自分を一つにさせる
  5. 深呼吸3回してすっきりした状態で目を開ける

10. 願望実現への応用

佳代：このイメージ力の訓練は願望実現にも有効。ただイメージするだけでなく、感情レベルまで味わうのがコツ。

体験談：車が欲しいと思った時、実際に車を手に入れてワクワクしている自分をイメージした。「やった！車が手に入った！」と感情レベルで味わった。1ヶ月後、実家から「車を処分するけど欲しいか」と連絡が来た。

感情レベルまで味わうと、宇宙の創造の原理にアクセスでき、現実化が早く起こる。
