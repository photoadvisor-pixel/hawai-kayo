第09回セミナー：エネルギーワーク（1）

イントロダクション

瞑想グッズについて

佳代：チーン（ティンシャの音）に乗っかるイメージで、音に意識を乗せて宇宙の中心に入っていく。

重要：自分の魂の波動に合った音を探すこと。ネットで買わず、実際にお店に行って「これいいよね」というものを選ぶ。石も同様。

自分を大切にすること

  • まず自分が整っていないと人のお世話はできない
  • 自己犠牲は続かない
  • 自分を丁寧に大切に扱うこと
  • 自分に居心地のいい空間を持つこと

佳代：瞑想で座る喜びを持って欲しい。瞑想すると色々分かってくるからワクワクが止まらなくなる。

コラム：旅先での波動対策（ホテルの枕）

ホテルはいろいろな人が泊まる場所。特に枕には、過去に寝た多くの人の想念が転写されていることがある。 - 対策： - 備え付けの枕は使わず、持参したタオルを3枚くらい巻いて枕代わりにする（佳代の実践法）。 - 自分の波動が染み込んだ瞑想用座布団を持参するのも良い。 - 部屋に入ったらまず光のシャワーイメージや音（ティンシャなど）で浄化する。 - ホテルの枕を使う場合は、使う前にしっかり光で浄化すること。

波動を読み取る訓練

練習1：自分のいる場所の波動を読み取る

目を閉じて3回深呼吸し、その場の空気やエネルギーを感じる。感じたままを書き出す。

受講生の回答例： - 雑ついている、落ち着いていない（ナイさん） - 澄んだ空気感、静けさ（けんさん） - 整っているところと整っていないところの差が激しい（いずちゃん） - 透き通っている、清々しい（まゆさん）

光の浄化瞑想

あまり良いエネルギーを感じない時は白か金色の光で浄化する。

手順： 1. 福式呼吸で瞑想状態に入る 2. 浄化したい部屋の上に大きな光の玉をイメージ 3. 光のシャワーで部屋全体を洗い流す 4. 汚れは全部地球の中心に返すイメージ 5. 綺麗にならない場合は壁・天井・床ごと外して地球の中心に返す 6. 宇宙から新しい部屋を受け取ってすっぽりはめる

ポイント：その部屋に愛を感じるかどうか。

参加者のユニークな浄化イメージ（ヒント）： - 金粉のシャワー：光だけでなく、金粉がドサッと降ってくるイメージ（まゆさん）。 - 完全リフォーム：汚れが落ちない場合は、壁も天井も一気に外して宇宙に返し、新しいピカピカの部屋を宇宙から受け取って「パコッ」とはめる（まゆさん）。 - 掃除ポイントの発見：浄化瞑想をすると、普段掃除が行き届いていない場所（隅の埃など）に気づくことがある（いずちゃん、ナエさん）。これは潜在意識からのサイン。

人の持ち物から波動を読み取る

練習3：誰かの持ち物の波動を感じる

目を閉じて、その物を手に取り、何を感じるか書き出す。

受講生の体験例： - 書道作品から「積極的に前に進むぞ」というエネルギー - 父の時計から「温かさ、包容力、大きさ」 - 母のポーチから「楽しい、心の温かさ」

自分自身を俯瞰する

練習4：地球の外から自分を見る

  1. 地球の外にいる自分をイメージ
  2. 美しい地球を見ながら、どんな心地か感じる
  3. 地球の中にいる自分を思い浮かべる
  4. 改善すべき点やアイデアがあれば感じ取る

受講生の気づき： - 部屋が散らかっている→物の位置を変えよう - 周りの空間をお気に入りにしたい→断捨離 - 時間に追われている→握りしめているものを手放す

クラスメートの波動を読み取る

互いの波動を読み取るワーク。3つくらいのワードで感じたことを伝え合う。

けんさんへの印象： - 凛としている、静寂（いずちゃん） - 少年の波動（まゆみさん） - 楽しい感じ（かずみさん） - 頭の中わちゃついてるイメージ（ナイさん）

いずちゃんへの印象： - 真面目、清（けんさん） - 冷静、燃えるものを感じる（ナイさん） - 内外の美を追求するエネルギー（まゆさん）

佳代：皆さんそれぞれに面白い。客観的に自分のことを聞くのは気づきになる。紙に書いて貼っておくといい。

様々な場所の波動を読み取る

練習：各地のエネルギーを感じ取る

考えず感覚的に、右脳で感じる。

例： - 東京：雑、ノイジー、パワフル - 大阪：ガチャガチャ、笑い、パワフル - 京都：品がある、裏表、伝統、歴史 - 北海道：広大 - 沖縄：開放、伝統民族 - 富士山：神聖、パワー - 国会議事堂：重厚、黒い嘘 - 東京ディズニーランド：夢、ワクワク - 伊勢神宮：神聖、守護 - ニューヨーク：カラフル、エンターテイメント - ハワイ・オアフ島：海、リゾート、癒し - マウイ島：悟り、宇宙、癒し、活力

佳代：マウイ島は人々の魂の進化を加速させる場所。意識の次元シフトの鍵を握る場所。

クレンジングとセルフヒーリング

目的

時代が大きく変わっている今、しっかりとした自分の軸（本当の自分）につながることが必須。波動を整えることが全てをうまく回していく鍵。

意識の世界は現実を生み出す設計図： - 意識がクリアで宇宙の根源に指していれば、魂レベルで納得のいく現実が生み出される - 自分にとっての最高のヒーラーは自分自身 - 人生のシナリオを書き換えるのも自分次第

波動チェックポイント

以下を感じたら要注意。クレンジングとヒーリングが必要なサイン。

  1. 気分が優れない（モヤモヤ、落ち込み、イライラ）
  2. 「モヤモヤラッキー」＝進化するタイミング
  3. 自分のことを心から好き・愛していると思えない
  4. オープンな心が持てず素直な気持ちでいられない
  5. ワクワクや時めき感、喜び、幸せを感じない
  6. 鏡で自分と向き合えない（目が笑っていない）
    • 鏡で自分の黒目をじーっと凝視してみる。
    • 変なものが入っている時のサイン：
      • 素直にニコッと笑えない。
      • ずっと見ていると口が歪んでくる、すごい形相に見える。
    • こういう時は無理に笑おうとせず、まずはクレンジング（浄化）が必要。

クレンジングの方法

  1. シャワーやお風呂に入る
  2. 自然の中で過ごす（週1回でも）
  3. 運動・食事（汗が滲む程度、オーガニック）
  4. グラウンディング

グラウンディング

地に足をつけ、安定した状態で日常生活を送るための技術。

方法1：地球に根を張る - 背骨を真っ直ぐにして座る - 地球の中心から赤いマグマのエネルギーが上がってくるイメージ - 自分の中心を通って頭頂から宇宙に突き抜ける - 宇宙の中心からの光が下りてきて地球の中心に突き抜ける

方法2：木になる - 自分が大きな木になったイメージ - 根が地球の中心まで伸びている - 枝葉が宇宙に向かって広がる

朝目覚めたら行い、寝る前にも行うのが理想。

今日のまとめ

  1. 波動を読み取る感覚は誰にでもある
  2. 自分の魂に合ったグッズを選ぶ
  3. 光の浄化瞑想で空間を整える
  4. 俯瞰することで改善点が見える
  5. モヤモヤは進化のサイン（モヤモヤラッキー）
  6. 日々のクレンジングとグラウンディングが大切
  7. 意識の世界が現実を生み出す設計図
