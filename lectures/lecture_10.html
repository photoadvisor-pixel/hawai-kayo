第10回セミナー：エネルギーワーク（2）

前回の復習

前回（第9回）ではエネルギーワークの基礎を学んだ。今回はその応用編として、セルフヒーリング、他者へのヒーリング、地球のヒーリングまで学ぶ。

セルフヒーリング：マインドからのアプローチ

1. 体をリラックスさせる

  • ヒーリング音楽、シータ波の音楽を聴く
  • 呼吸法
  • アロマの香り
  • セルフマッサージ

注意：人にマッサージを受ける場合は要注意。施術者の波動が低いと、こちらがエネルギーを受け取ってしまう場合がある（お金を払って相手のヒーリングをしていた経験あり）。

対策： - どうしても受ける場合は、施術前に心の中で相手と自分のハイヤーセルフがつながるように意図する。 - 相手を光でクレンジングし、「この人の最高の能力が発揮されますように」「最も良い形で施術が行われますように」とお願いしてから受けると良い。

2. 自分に素直になる

素直になればなるほど霊的進化が早い。オープンマインドで宇宙にハートを開く。

3. 自分の心の声に耳を傾ける

体の不調がある場合は体の声に耳を傾ける。

佳代：「なぜここが痛いの？」「どうしたいの？」と体に聞くと答えが来る。対話をしていく。

  1. 過去の自分（インナーチャイルド）との対話
    • 「インナーチャイルド」は幼い子供だけではない。
    • 例えば20代で大失恋した時の痛みなら、その時の「20代の自分」をイメージして対話する。
    • その時点の自分に戻り、当時の相手（彼など）と会話させて、言いたかったことを言わせてあげる。
    • 涙が溢れてきたら止めない。文章や絵に吐き出すことも有効。

コラム：瞑想中のビジョン（マハリシの導き）

佳代自身も、人間生活で色々ありモヤモヤしている時に、瞑想で砂地を不安定に歩いているビジョンを見た。 その時、師であるマハリシや聖者たちが「なぜそこを歩いているの？こっち（覚醒側）でしょ？」と導いてくれた。 人間である以上、波動が落ちることはある。それに気づき、再び本来の位置（覚醒・愛の意識）に戻ることが大切。

5. 体験からの学びを見つける

物事を表面から見ず、背後の宇宙メッセージを見る。すると感謝に変わり、解放される。

カルマの解消と意識の次元シフト

  • 3次元から4次元（アセンション）は最もハード
  • 4次元から5次元はそれほど難しくない
  • アファメーションの力が有効
  • 5次元意識＝無条件の愛に目覚める

セルフヒーリング：イメージからのアプローチ

1. 太陽の光のボディ

  1. 福式呼吸をする
  2. 頭の上に太陽をイメージ
  3. 全身を光のシャワーで洗い流す
  4. 光を吸って黒い霧を吐く
  5. 60兆の細胞がニコニコ光輝くイメージ
  6. 自分が太陽そのものになる

佳代の体験：この呼吸法を毎日30分、3日続けたら高熱が出た。それだけパワフルな浄化力がある。

2. 背骨を光り輝かせる

  1. 地球の中心から無数の白い光が放射されている
  2. その1本が微底骨から上昇し、背骨を通って頭の上まで
  3. さらに宇宙の中心まで上昇
  4. 宇宙の中心から白い光が降りてくる
  5. 頭頂から背骨を通って地球の中心まで
  6. 自分が真っ白に光輝く柱になる

効果：一気にチャクラを活性化できる最も安全な方法。

3. チャクラの活性化

7つのチャクラ：

チャクラ 場所 色 テーマ
第1 微底骨 赤 生存欲求、グラウンディング
第2 仙骨/へその下 オレンジ 感情表現、創造性
第3 溝落ち 黄 自己意識、意思、力
第4 胸の中央 緑 愛、思いやり、調和
第5 喉 水色 自己表現、コミュニケーション
第6 眉間 藍 直感、洞察、知恵
第7 頭頂 紫 高次の意識、悟り

注意：チャクラを変にいじると危険。必ずグラウンディングコードを地球の中心につないでから行う。

チャクラのクレンジング方法： - 方法1：チャクラをビーカーに入れてシャカシャカ洗うイメージ - 方法2：背骨を光り輝かせる方法で一気に活性化（最も安全）

丹田エネルギーチャージ

ハワイ島の火の女神ペレから受け取ったワーク。

手順

  1. 地球の中心に入っていくイメージ
  2. 地球のエネルギー（溶岩、海、大陸など）を全身で感じる
  3. 地球の中心から太い赤/オレンジのエネルギーが上昇
  4. 足の裏からぐんぐん上がって丹田まで
  5. 頭の上から金色のエネルギーが降りてくる
  6. 丹田で2つのエネルギーが融合して大きな光の玉になる
  7. 全身にエネルギーが満ちていく

効果：活力溢れる生命エネルギー。夢や目標の現実化をサポート。

プロテクト

方法1：自分だけの太陽

  1. 胸の中心から光の種を取り出す（ハイヤーセルフのエッセンス）
  2. 中空に投げて自分だけの太陽を作る
  3. 太陽を大きく広げてすっぽり入る
  4. 自分が太陽そのものになる

原理：太陽は光を放射するが負のエネルギーは吸収しない。

方法2：六芒星のプロテクト（最強）

  1. イメージで円を描く
  2. 円の周りにびっしり光の柱を立てる
  3. 円の中心に六芒星（ダビデの紋章）をイメージ
  4. 六芒星の中心に水晶玉をイメージ
  5. 自分がその水晶玉の中に入る
  6. 水晶玉の周りにもびっしり光の柱を立てる
  7. （詳細）六芒星の6つの頂点それぞれにも光の柱を立て、さらに中心の自分の周りにも立てるイメージ。

佳代：プーケットのイベント前に「エネルギー的に嫌だな」と感じた際、宇宙からこの方法が降りてきた。六芒星は進化と合一の象徴（アートマン）。これで完全にネガティブなエネルギーが遮断され、イベントも無事に終えられた。 ポイント：「できている」と確信を持ってイメージすることが重要。イメージはエネルギーフィールドに青写真として焼き付く。

佳代：プーケットのイベント前にこの方法が降りてきた。六芒星は進化と合一の象徴。これで完全にネガティブなエネルギーが遮断される。

他者へのヒーリング

基本的な手順

  1. 自分をクレンジングする
  2. 宇宙の根源の光と一体化する
  3. 左手の手のひらに相手のミニチュアをイメージ
  4. 右手で上から覆う
  5. ハイヤーセルフにお願いして必要な分だけ受け取ってもらう

重要：エゴを入れない。「こうしてあげたい」は不要。自分は宇宙のエネルギーを渡す「優秀な管」になるだけ。

佳代：本物のヒーラーはエゴを入れない。精神的にも霊的にも成熟していることが必要。「治してあげよう」という思いすらエゴになり得る。ただ純粋なパイプとなり、宇宙の無限の愛を流すだけ。

実践エピソード（なおこさんへのヒーリング）： セミナー中、体調不良（発熱）の参加者なおこさんに全員でヒーリングを実施。 - 結果：なおこさんは「元気玉をもらったみたい」と回復を実感。 - 送る側の体験：参加者（まゆみさん、いずちゃんなど）の手がものすごく熱くなった。これは宇宙のエネルギーが流れた証拠。純粋なパイプになれば、誰でもヒーリングは可能。

エネルギー調整

物のエネルギーを変える

  1. その物にグラウンディングコードをつなぐ
  2. 古いエネルギーを地球の中心に返す
  3. 宇宙の中心からピカピカの光の玉を届ける
  4. 物と光の玉を一体化させる

部屋のエネルギーを変える

  1. 部屋全体を天井から丸ごと剥がして地球の中心に返す
  2. 宇宙の中心から新しい金色に輝く部屋を作る

建物のエネルギーを変える

大きな建物はミニチュアにして手のひらに置き、ヒーリングと同じ要領で行う。

地球のヒーリング

  1. 地球をイメージする
  2. 地球の外のところから俯瞰する
  3. 両手で地球を包み込む
  4. 宇宙の根源から届いたエネルギーを送る
  5. 「必要な分だけ」受け取ってもらう

戦争、飢餓など解決したい問題に対して「そのような問題が解決した地球」をイメージして送る。

ホテルの部屋の浄化

  1. チーン（ティンシャ）で浄化
  2. パンパンパンと手を叩いて浄化
  3. エッセンシャルオイルを撒く
  4. 光のシャワーで浄化

重要：枕は特に注意。様々な人の念が転写されている。自分の枕を持参するか、光で浄化してから使う。
